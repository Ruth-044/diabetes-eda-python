# Diabetes Data Analysis (EDA)

## Overview

This project explores a diabetes dataset using Python for data cleaning, exploratory data analysis (EDA), and visualization.

## Objectives

* Handle missing and invalid clinical values
* Explore relationships between variables
* Visualize glucose and blood pressure trends

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Data Cleaning

* Replaced invalid zero values with missing values
* Imputed missing values using median
* Removed duplicate entries

## Visualizations

* Scatter plot with regression line
* Glucose distribution histogram
* Correlation heatmap

## Key Findings

* Glucose levels show a right-skewed distribution
* Weak positive relationship observed between glucose and blood pressure
* Several clinical variables show moderate correlation

## Files

* diabetes-data-analysis-with-python-eda.ipynb
* diabetes.csv

## Conclusion

This project demonstrates foundational healthcare data analysis skills using Python and exploratory visualization techniques.
